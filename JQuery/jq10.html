a要素の機能を無効化する方法です

click()は任意の要素を指定できますが、元々クリックされる要素であるaタグをクリックすると期待していない動作をされます。

回避方法2

scriptで回避します無名関数内部でreturn falseを使います。

画像を変更 

<a href="./images/ferret.jpg">画像を変更</a>
<p><img src="./images/kani1.jpg" alt="カニ1"></p>

<script>
	$(function () {
		$("a").click(function () {
			$("img").attr("src", "./images/kani2.jpg").attr("alt", "カニ２");
			return false;
		});
	});
</script>
